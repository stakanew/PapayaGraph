# Papaya Graph

OSINT-инструмент с 3D-графом (как Maltego + Obsidian).

> ⚠️ Проект создан с помощью ИИ (AI-generated).  
> Система друзей и чат работают нестабильно и находятся в доработке.

## Быстрый старт
1. Установи Python 3.10+
2. Клонируй репозиторий
3. Запусти `start.bat` (Windows) или `python main.py`

## Возможности
- 3D-граф связей
- Автоопределение данных при вставке (Ctrl+V)
- Трансформы для OSINT
